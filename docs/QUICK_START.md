# Quick Start Guide

Welcome to your College Application Tracker! This guide will help you get started quickly.

## Installation & Setup

### Step 1: Install Dependencies
Open your terminal in the project folder and run:
```bash
npm install
```

### Step 2: Start the Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
Navigate to `http://localhost:5173` in your web browser.

## Your First Steps

### 1. Add Your First University
1. Click the **"Add University"** button on the dashboard
2. Fill in the required information:
   - University name
   - Location
   - Region
   - School size
   - Application deadline
3. Add any notes or early decision deadlines
4. Click **"Add University"**

### 2. Create a Checklist
1. Click on a university card to view details
2. Go to the **"Checklist"** tab
3. Click **"Add Task"**
4. Add tasks like:
   - Request transcripts
   - Complete application essays
   - Submit test scores
   - Request recommendation letters
5. Check off tasks as you complete them

### 3. Set Up Reminders
1. Navigate to **"Reminders"** in the top menu
2. Click **"New Reminder"**
3. Fill in:
   - Title (e.g., "Submit Harvard Application")
   - Description
   - Due date
   - Priority level
   - Type (deadline, interview, follow-up, task)
4. Link it to a specific university if applicable

### 4. Create Email Templates
1. Go to **"Communications"**
2. Click **"New Template"**
3. Create templates for common scenarios:
   - Requesting recommendation letters
   - Following up with admissions
   - Thank you emails after interviews
4. Use the **"Copy"** button to quickly use templates

### 5. Track Your Progress
1. Visit the **"Analytics"** page
2. See your:
   - Total universities
   - Completion rate
   - Status distribution
   - Upcoming deadlines
3. Use this to stay motivated and on track!

## Key Features to Remember

### Dashboard
- **Search**: Use the search bar to find universities quickly
- **Filters**: Filter by region, size, or status
- **Sorting**: Sort by deadline to prioritize urgent applications
- **Alerts**: Watch for red (overdue) and yellow (urgent) warnings

### University Details
- **Tabs**: Switch between Overview, Checklist, Essays, and Interviews
- **Status Dropdown**: Update application status as you progress
- **Auto-save**: Notes are automatically saved when you type
- **Word Counter**: Essays show real-time word count

### Communications
- **Templates Tab**: Store reusable email templates
- **History Tab**: Log all communications with universities
- **Follow-up Tracking**: Mark communications that need follow-up
- **Search**: Find past communications easily

### Reminders
- **Priority Badges**: High (red), Medium (yellow), Low (green)
- **Filters**: View all, active, or completed reminders
- **Visual Alerts**: Overdue and urgent reminders are highlighted
- **Quick Complete**: Check the box to mark as complete

### Analytics
- **Real-time Stats**: Updates automatically as you add/update data
- **Visual Bars**: See progress at a glance
- **Motivational Messages**: Get encouragement based on your progress
- **Deadline Preview**: See upcoming deadlines for the next 30 days

## Pro Tips

### Organization
1. **Add all universities first** - Even if you haven't started the application
2. **Create checklists immediately** - Break down each application into steps
3. **Set reminders for everything** - Especially for deadlines 2 weeks out
4. **Log every communication** - You'll thank yourself later

### Workflow
1. **Morning Routine**: Check reminders and upcoming deadlines
2. **Weekly Review**: Visit Analytics to see overall progress
3. **After Each Interaction**: Log it in Communications
4. **Before Bed**: Update checklist items and notes

### Staying Motivated
- Check the Analytics page regularly to see your progress
- Celebrate each completed checklist item
- Use the completion rate as a game - try to increase it!
- Read the motivational messages at the bottom of Analytics

## Common Questions

**Q: Where is my data stored?**
A: All data is stored locally in your browser using localStorage. It's private and never leaves your device.

**Q: Can I access this from multiple devices?**
A: Currently, data is device-specific. Consider keeping important notes backed up externally.

**Q: What happens if I clear my browser data?**
A: Your application data will be lost. Be sure to back up important information regularly.

**Q: Can I export my data?**
A: Not currently, but you can copy and paste important information to a document for backup.

**Q: The app won't load, what do I do?**
A: Try:
1. Refreshing the page (Cmd+R or Ctrl+R)
2. Clearing browser cache
3. Checking the browser console for errors (F12)
